# 📄 Sign-up Form | The Odin Project

Este projeto é uma implementação de um formulário de cadastro fictício, desenvolvido como parte do curso **Intermediate HTML and CSS** da plataforma The Odin Project.

O objetivo principal foi praticar conceitos fundamentais de **HTML semântico**, **CSS moderno**, além de trabalhar com layout e estilização de formulários.

## 🚀 Visão Geral

O projeto consiste em uma página de cadastro com:

* Uma **seção lateral com imagem de fundo**
* Um **logo personalizado**
* Um **formulário estruturado e estilizado**
* Feedback visual para inputs (focus e validação)

---

## 🎯 Objetivos do Projeto

* Praticar estruturação com **HTML5**
* Trabalhar com **Flexbox** para layout
* Aplicar estilos em formulários
* Utilizar pseudo-classes como:

  * `:focus`
  * `:invalid`
* Importar e utilizar **fontes externas**
* Organizar melhor o CSS

---

## 🧠 O que foi aprendido

Durante o desenvolvimento, foram reforçados conceitos importantes como:

* Organização de layout com `display: flex`
* Uso de `background-image` com `cover`
* Criação de formulários acessíveis
* Uso correto de `label` + `input`
* Controle visual de estados dos inputs
* Uso de `@font-face` para fontes customizadas
* Hierarquia e separação de responsabilidades no CSS

## 🛠️ Tecnologias Utilizadas

## 🎨 Destaques da Interface

* Layout dividido em duas áreas (imagem + formulário)
* Efeito de foco nos inputs (borda azul + sombra)
* Validação visual para senha inválida (borda vermelha)
* Botão estilizado com cor baseada no design (#596D48)
* Overlay escuro no logo para melhorar a legibilidade
